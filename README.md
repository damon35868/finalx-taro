<h1 align="center">FinalX</h1>
<p align="center">基于Taro、React打造的一款跨端开发的工具及组件库</p>

## 使用 （[详细文档](https://finalx.vercel.app)）

### Installation 安装

#### npm

```shell
npm install @finalx/cli -g
```

### Init 生成项目结构

```shell
finalx init <projectName>
```
